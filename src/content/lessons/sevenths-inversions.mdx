## The bass changes the chord's posture

An inversion keeps the same chord tones but changes which tone sits lowest. C E G is C major in root position. E G C is still C major, but the third is now in the bass.

## Seventh chords add one more stacked third

A seventh chord places one more chord tone above the fifth. G7 contains G B D F: a G major triad plus F natural.
