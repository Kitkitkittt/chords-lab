## The concept chain

This course starts with named pitch, places it on staff and keyboard views, gives it rhythm, changes it with accidentals, organizes it into scales, measures it with intervals, and stacks it into triads.

<Callout title="A useful review sentence" tone="harmony">
  Notes become scales; distances become intervals; stacked intervals become chords.
</Callout>

## What to review first

If a lesson feels heavy, move backward to the first unclear link. Chord spelling usually depends on interval spelling. Interval spelling usually depends on note letters and accidentals.

## What comes later

Later releases can add seventh chords, Roman numerals, cadences, harmonic function, dictation, melody writing, form, counterpoint, figured bass, pop/rock harmony, and post-tonal basics.
